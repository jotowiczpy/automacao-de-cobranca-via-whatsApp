# automacao-de-cobranca-via-whatsApp
utomação em Python para envio de mensagens de cobrança pelo WhatsApp utilizando uma planilha do Excel como base de clientes.  O script realiza a leitura da planilha, envia mensagens personalizadas para clientes com status pendente, identifica números que não possuem WhatsApp e atualiza automaticamente o status de cada contato.
